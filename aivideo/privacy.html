隐私政策

最后更新：2026-06-01

本政策说明「AI财知道」相关产品（含通过 TikTok for Developers API 发布内容的功能）如何收集、使用与保护信息。

1. 我们收集的信息

  • TikTok 授权信息：当您通过 OAuth 连接 TikTok 账号时，我们仅获取 TikTok 开放平台授予的权限范围内数据（如基本账号信息 user.info.basic，以及用于上传/发布视频的 video.upload / video.publish 权限）。我们不会索取超出授权范围的权限。
  • 发布内容：您选择发布的视频文件、标题/描述/标签及您选择的隐私与互动设置，会经 TikTok Content Posting API 传输至 TikTok 服务器处理。
  • 本地凭证：OAuth 访问令牌与刷新令牌仅存储在您本机或您自行部署环境中的加密凭据目录，不上传至第三方营销平台。

2. 信息用途

  • 代表已授权账号向 TikTok 上传并发布视频；
  • 在发布前查询创作者可用隐私级别与互动能力（creator_info/query）；
  • 维护授权会话有效性（刷新 token）。

3. 与 TikTok 的关系

本功能使用 TikTok for Developers 官方 API。视频托管、展示与 TikTok 侧数据处理受 TikTok 服务条款 与 TikTok 隐私政策 约束。我们不会在未经您操作的情况下向 TikTok 发布内容。

4. 共享与披露

除向 TikTok 按 API 要求传输发布所需数据外，我们不会出售您的个人数据。法律要求或为保护权利所必需时除外。

5. 数据保留与安全

本地 token 可由您随时通过重新授权或删除凭据文件撤销。请妥善保管设备与 credentials/tiktok/ 目录。

6. 您的权利

您可在 TikTok App 内管理已授权应用、撤销授权、删除已发布内容。如需停止使用 API 发布，请断开本应用授权并删除本地 token。

7. 联系我们

有关本政策或 TikTok 集成的疑问，请通过您提交 TikTok 开发者应用审核时填写的联系邮箱与我们联系。
