服务条款

最后更新：2026-06-01

使用「AI财知道」及相关发布工具（含 TikTok Content Posting API 集成）即表示您同意本条款。

1. 服务说明

本服务帮助内容创作者将自制财经解说视频发布至已授权的 TikTok 账号。服务依赖 TikTok 开放平台，其可用性、审核状态与功能限制以 TikTok 官方为准。

2. 用户责任

  • 您须拥有对所发布内容的合法权利，并遵守 TikTok 社区准则与当地法律；
  • 不得利用本服务发布误导性投资建议、荐股喊单或违规营销内容；
  • 财经内容须包含「不构成投资建议」等适当提示（由您在发布文案中自行落实）。

3. TikTok API 使用

您授权我们代您调用 TikTok API 时，须遵守 TikTok Developer Guidelines 及 Content Posting API 产品规范，包括但不限于：发布前展示创作者信息、由您选择隐私级别（无默认预选）、展示音乐使用确认及商业内容披露选项（如适用）。

4. 免责声明

内容仅供学习交流，不构成任何投资建议。我们不对因使用或无法使用本服务、TikTok 审核/限流/封号等导致的损失承担责任。

5. 终止

您可随时撤销 TikTok 授权并停止使用。我们可因违反条款或平台政策而停止提供集成支持。

6. 条款变更

我们可能更新本条款，更新后于本页公布生效日期。

7. 联系

问题请通过 TikTok 开发者应用登记的联系渠道与我们联系。
